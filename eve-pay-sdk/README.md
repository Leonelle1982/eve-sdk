# EVE Pay SDK (v1)

Plug-and-play payment API for ERP, Redmine, commerce, finance, and
sustainability systems. Platform-agnostic and blockchain optional.

## Base URL
https://api.evegrocer.app/v1

## Core Endpoints
POST /payments/init
POST /payments/confirm
